Instalación y Configuración en macOS

En esta sección aprenderás a instalar y configurar Claude Code en tu Mac paso a paso. Cubriremos tres métodos de instalación y la configuración completa.

Método 1: Instalación con Homebrew (Recomendado)

Homebrew es el gestor de paquetes más popular para macOS. Este es el método más simple y recomendado.

Paso 1: Instalar Homebrew (si no lo tienes)

# Verificar si Homebrew está instalado
which brew

# Si no está instalado, ejecutar:
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
🍺

Configuración Post-Instalación de Homebrew

Después de instalar Homebrew, es posible que necesites agregarlo a tu PATH. En macOS con Apple Silicon (M1/M2/M3), ejecuta:

# Para Apple Silicon (M1/M2/M3)
echo 'eval "$(/opt/homebrew/bin/brew shellenv)"' >> ~/.zshrc
source ~/.zshrc

# Para Intel Macs
echo 'eval "$(/usr/local/bin/brew shellenv)"' >> ~/.zshrc
source ~/.zshrc

Paso 2: Instalar Claude Code

# Actualizar Homebrew
brew update

# Instalar Claude Code
brew install anthropic/tap/claude

# Verificar la instalación
claude --version

Método 2: Instalación con npm

Si prefieres usar npm (Node Package Manager), puedes instalar Claude Code como paquete global.

Prerrequisitos

# Verificar Node.js (versión 18 o superior)
node --version

# Si no tienes Node.js, instálalo con Homebrew
brew install node

Instalación

# Instalar Claude Code globalmente
npm install -g @anthropic-ai/claude

# Verificar instalación
claude --version

Método 3: Instalación desde Binarios

Para instalación manual o si los métodos anteriores no funcionan:

# Descargar el binario para macOS
curl -L https://github.com/anthropics/claude/releases/latest/download/claude-macos-arm64.tar.gz -o claude.tar.gz

# Para Intel Macs, usar:
# curl -L https://github.com/anthropics/claude/releases/latest/download/claude-macos-x64.tar.gz -o claude.tar.gz

# Extraer
tar -xzf claude.tar.gz

# Mover a directorio en el PATH
sudo mv claude /usr/local/bin/

# Hacer ejecutable
sudo chmod +x /usr/local/bin/claude

# Verificar instalación
claude --version

Configuración Inicial

1. Obtener API Key de Anthropic

Antes de usar Claude Code, necesitas una API key:

  1. Visita console.anthropic.com
  2. Crea una cuenta o inicia sesión
  3. Ve a "API Keys" en el menú lateral
  4. Haz clic en "Create Key"
  5. Copia la key (empieza con sk-ant-...)
⚠️

Seguridad de la API Key

Nunca compartas tu API key ni la subas a repositorios públicos. Guárdala de forma segura y rótala regularmente.

2. Configurar Variables de Entorno

Configura tu API key en el archivo de configuración de tu shell:

# Abrir el archivo de configuración de zsh
nano ~/.zshrc

# O si usas bash:
# nano ~/.bash_profile

# Agregar estas líneas al final:
export ANTHROPIC_API_KEY="sk-ant-tu-api-key-aqui"
export CLAUDE_CODE_MODEL="claude-3-sonnet-20240229"  # Opcional

# Guardar (Ctrl+O, Enter, Ctrl+X) y recargar:
source ~/.zshrc

3. Inicialización de Claude Code

# Primera ejecución - configuración interactiva
claude init

# Esto creará el directorio de configuración en:
# ~/.config/claude/

4. Archivo de Configuración

Claude Code usa un archivo de configuración en ~/.config/claude/config.json:

{
  "apiKey": "${ANTHROPIC_API_KEY}",
  "model": "claude-3-sonnet-20240229",
  "maxTokens": 4096,
  "temperature": 0.7,
  "contextWindowSize": 100000,
  "editor": "code",
  "theme": "dark",
  "autoSave": true,
  "historyLimit": 50,
  "cache": {
    "enabled": true,
    "ttl": 3600,
    "maxSize": "500MB"
  }
}

Verificación de la Instalación

Ejecuta estos comandos para verificar que todo está funcionando:

# Ver versión
claude --version

# Ver configuración actual
claude config --show

# Prueba simple
claude "Hola, ¿estás funcionando correctamente?"

# Ver ayuda
claude --help

Integración con Editores

Visual Studio Code

# Configurar VS Code como editor predeterminado
claude config set editor "code"

# Abrir archivo con Claude Code desde VS Code
# En VS Code, abre la paleta de comandos (Cmd+Shift+P) y busca
# "Terminal: Create New Terminal"

# Luego puedes usar:
claude "Explica este archivo" archivo.js

Cursor Editor

# Configurar Cursor como editor
claude config set editor "cursor"

Configuración Avanzada

Alias de Terminal

Crea alias para comandos frecuentes:

# Agregar a ~/.zshrc
alias cc="claude"
alias ccr="claude review"
alias cct="claude test"
alias ccd="claude doc"

# Recargar
source ~/.zshrc

# Ahora puedes usar:
cc "Genera una función factorial"

Configurar Proxy (si es necesario)

# Agregar a ~/.zshrc
export HTTPS_PROXY="http://proxy.company.com:8080"
export HTTP_PROXY="http://proxy.company.com:8080"
