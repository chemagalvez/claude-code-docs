Comandos Básicos y Flujo de Trabajo

Sintaxis General

Claude Code sigue una sintaxis consistente para todos los comandos:

claude [opciones] [comando] [argumentos]

Comandos Esenciales

1. Modo Interactivo

# Iniciar sesión interactiva
claude

# Iniciar con un archivo de contexto
claude -f src/app.js

# Iniciar con múltiples archivos
claude -f src/*.js

2. Consultas Directas

# Pregunta simple
claude "¿Cómo implemento un singleton en TypeScript?"

# Analizar archivo
claude "Explica este código" utils.ts

# Generar código
claude "Crea una función que valide emails" > validators.js

3. Generación de Código

# Generar con especificaciones
claude generate "API REST con Express, incluye rutas CRUD para usuarios"

# Generar y guardar
claude generate "Componente React de formulario de login" > LoginForm.tsx

# Generar tests
claude test src/calculator.js

4. Revisión de Código

# Revisar archivo
claude review src/index.js

# Revisar con focus específico
claude review --focus security auth.js

# Revisar cambios de git
git diff | claude review

5. Refactorización

# Refactorizar archivo
claude refactor legacy-code.js

# Refactorizar aplicando patrón
claude refactor --pattern "factory" ProductManager.js

# Refactorizar a TypeScript
claude refactor --to typescript app.js

Opciones Globales

Opción Descripción Ejemplo
--model Especificar modelo de Claude --model opus
--file, -f Archivo(s) de contexto -f src/app.js
--output, -o Archivo de salida -o result.js
--verbose, -v Modo detallado -v
--json Salida en JSON --json
--help, -h Mostrar ayuda -h

Flujo de Trabajo Típico

💡

Workflow Recomendado

Este es un flujo de trabajo típico para desarrollar una nueva feature:

Paso 1: Planificación

# Diseñar arquitectura
claude "Diseña la arquitectura para un sistema de autenticación JWT \
incluyendo registro, login, refresh tokens y middleware"

Paso 2: Implementación

# Generar estructura base
claude generate "Implementa el sistema de autenticación diseñado"

# Generar archivos específicos
claude "Crea el modelo de Usuario con Mongoose" > models/User.js
claude "Crea el controlador de autenticación" > controllers/auth.js
claude "Crea middleware de verificación JWT" > middleware/auth.js

Paso 3: Testing

# Generar tests
claude test controllers/auth.js > tests/auth.test.js

# Revisar cobertura
claude "Analiza la cobertura de tests y sugiere tests adicionales" \
tests/auth.test.js

Paso 4: Documentación

# Generar documentación
claude doc controllers/auth.js

# Crear README
claude "Genera un README para el módulo de autenticación" > auth/README.md

Paso 5: Revisión

# Revisar todo el código
claude review src/auth/**/*.js

# Verificar seguridad
claude review --focus security src/auth/**/*.js

Atajos de Productividad en macOS

Aliases Útiles

# Agregar a ~/.zshrc
alias cc="claude"
alias ccg="claude generate"
alias ccr="claude review"
alias cct="claude test"
alias ccd="claude doc"
alias ccf="claude refactor"

# Función helper para revisar git diff
function ccgd() {
    git diff | claude review
}

# Función para generar y abrir en editor
function ccge() {
    claude generate "$@" | tee output.tmp && code output.tmp
}

Integración con Git

# Revisar cambios antes de commit
git diff | claude "Resume estos cambios en un mensaje de commit"

# Generar mensaje de commit
git diff --cached | claude "Genera un mensaje de commit convencional"

# Analizar conflictos de merge
git diff --name-only --diff-filter=U | xargs claude \
"Ayúdame a resolver estos conflictos de merge"

Ejemplos Prácticos

Debugging

# Analizar stack trace
claude "Analiza este error y sugiere soluciones" error.log

# Encontrar bug
claude "Este código no funciona correctamente. ¿Cuál es el problema?" \
buggy-code.js

Optimización

# Optimizar rendimiento
claude "Optimiza este código para mejor rendimiento" slow-function.js

# Reducir complejidad
claude "Simplifica esta función manteniendo la funcionalidad" \
complex-logic.js

Conversión entre Lenguajes

# JavaScript a TypeScript
claude "Convierte esto a TypeScript con tipos apropiados" app.js > app.ts

# Python a JavaScript
claude "Convierte este código Python a JavaScript moderno" script.py > script.js

Comando Qué hace
