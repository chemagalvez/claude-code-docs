Comandos Básicos y Flujo de Trabajo

Sintaxis General

Claude Code sigue una sintaxis consistente para todos los comandos:

claude-code [opciones] [comando] [argumentos]

Comandos Esenciales

1. Modo Interactivo

# Iniciar sesión interactiva
claude-code

# Iniciar con un archivo de contexto
claude-code -f src/app.js

# Iniciar con múltiples archivos
claude-code -f src/*.js

2. Consultas Directas

# Pregunta simple
claude-code "¿Cómo implemento un singleton en TypeScript?"

# Analizar archivo
claude-code "Explica este código" utils.ts

# Generar código
claude-code "Crea una función que valide emails" > validators.js

3. Generación de Código

# Generar con especificaciones
claude-code generate "API REST con Express, incluye rutas CRUD para usuarios"

# Generar y guardar
claude-code generate "Componente React de formulario de login" > LoginForm.tsx

# Generar tests
claude-code test src/calculator.js

4. Revisión de Código

# Revisar archivo
claude-code review src/index.js

# Revisar con focus específico
claude-code review --focus security auth.js

# Revisar cambios de git
git diff | claude-code review

5. Refactorización

# Refactorizar archivo
claude-code refactor legacy-code.js

# Refactorizar aplicando patrón
claude-code refactor --pattern "factory" ProductManager.js

# Refactorizar a TypeScript
claude-code refactor --to typescript app.js

Opciones Globales

Opción Descripción Ejemplo
--model Especificar modelo de Claude --model opus
--file, -f Archivo(s) de contexto -f src/app.js
--output, -o Archivo de salida -o result.js
--verbose, -v Modo detallado -v
--json Salida en JSON --json
--help, -h Mostrar ayuda -h

Flujo de Trabajo Típico

💡

Workflow Recomendado

Este es un flujo de trabajo típico para desarrollar una nueva feature:

Paso 1: Planificación

# Diseñar arquitectura
claude-code "Diseña la arquitectura para un sistema de autenticación JWT \
incluyendo registro, login, refresh tokens y middleware"

Paso 2: Implementación

# Generar estructura base
claude-code generate "Implementa el sistema de autenticación diseñado"

# Generar archivos específicos
claude-code "Crea el modelo de Usuario con Mongoose" > models/User.js
claude-code "Crea el controlador de autenticación" > controllers/auth.js
claude-code "Crea middleware de verificación JWT" > middleware/auth.js

Paso 3: Testing

# Generar tests
claude-code test controllers/auth.js > tests/auth.test.js

# Revisar cobertura
claude-code "Analiza la cobertura de tests y sugiere tests adicionales" \
tests/auth.test.js

Paso 4: Documentación

# Generar documentación
claude-code doc controllers/auth.js

# Crear README
claude-code "Genera un README para el módulo de autenticación" > auth/README.md

Paso 5: Revisión

# Revisar todo el código
claude-code review src/auth/**/*.js

# Verificar seguridad
claude-code review --focus security src/auth/**/*.js

Atajos de Productividad en macOS

Aliases Útiles

# Agregar a ~/.zshrc
alias cc="claude-code"
alias ccg="claude-code generate"
alias ccr="claude-code review"
alias cct="claude-code test"
alias ccd="claude-code doc"
alias ccf="claude-code refactor"

# Función helper para revisar git diff
function ccgd() {
    git diff | claude-code review
}

# Función para generar y abrir en editor
function ccge() {
    claude-code generate "$@" | tee output.tmp && code output.tmp
}

Integración con Git

# Revisar cambios antes de commit
git diff | claude-code "Resume estos cambios en un mensaje de commit"

# Generar mensaje de commit
git diff --cached | claude-code "Genera un mensaje de commit convencional"

# Analizar conflictos de merge
git diff --name-only --diff-filter=U | xargs claude-code \
"Ayúdame a resolver estos conflictos de merge"

Ejemplos Prácticos

Debugging

# Analizar stack trace
claude-code "Analiza este error y sugiere soluciones" error.log

# Encontrar bug
claude-code "Este código no funciona correctamente. ¿Cuál es el problema?" \
buggy-code.js

Optimización

# Optimizar rendimiento
claude-code "Optimiza este código para mejor rendimiento" slow-function.js

# Reducir complejidad
claude-code "Simplifica esta función manteniendo la funcionalidad" \
complex-logic.js

Conversión entre Lenguajes

# JavaScript a TypeScript
claude-code "Convierte esto a TypeScript con tipos apropiados" app.js > app.ts

# Python a JavaScript
claude-code "Convierte este código Python a JavaScript moderno" script.py > script.js
